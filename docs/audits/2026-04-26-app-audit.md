# PuckIQ App Audit — 2026-04-26

**Audited by**: Claude (full-app-audit skill, 3-wave build)
**Scope**: Every visible surface across 5 tabs + 2 hidden + 9 modals + onboarding entry, in both signed-out and gated states. Premium-only surfaces audited from code (RevenueCat sandbox not configured).
**Trigger**: "the app is honestly all over the place and I need to figure out how to make this app amazing" + "I don't want there to be articles, I want it to be stats focused for the hockey enthusiast"

## Surface area inventory

- **Tabs (5 visible)**: Today (`/`), My Team (`/myteam`), Players (`/players`), Explore (`/stats`), Hub (`/hub`)
- **Hidden routes (2)**: Teams (`/teams`, embedded in Explore), Models (`/models`, embedded in Explore)
- **Modals/sheets (9)**: PlayerDetailModal, TeamModal, StatTooltip, Team Picker A, Team Picker B, ModelEditScreen (full-screen), DataSeedingModal, PaywallModal (PremiumGate), RosterBuilder
- **Dashboard modules (6)**: StartSit, Trending, Alerts, WaiverWire, MatchupEdge, DailyInsight
- **Onboarding flow**: Triggered when `puckiq_onboarding_complete` is unset

## Vision

**[Stat Sheet](../design/vision.md)** — replaces "Rink Glass" (which was approved 2026-04-05 but only ~70% adopted; too abstract a metaphor to drive concrete decisions). Stat Sheet is opinionated, terse, and stat-forward: a daily briefing for the hockey enthusiast.

## What was wrong (root causes)

| # | Root cause | Symptom |
|---|------------|---------|
| 1 | **Three competing palettes** in `constants/theme.ts` | Saturated blue Explore panels (`theme.card = #192e5e`) clashing with dark Today modules (`rinkGlass.boards = #141829`) |
| 2 | **Module accents duplicated as decoration** | Pink stripe on Trending, orange on Daily Insight, green on Start/Sit, yellow on Alerts — six modules, six unrelated accent colors |
| 3 | **Editorial voice everywhere** | "ride the home ice edge", "Be the first on the leaderboard!", "Tough goalie matchup — model is split", "Make picks to build data!" |
| 4 | **Emojis in a system that explicitly bans emojis** | 🔥 fire bars in Trending, ⚡ in ShareablePickCard, 🔥 in Leaderboard streaks |
| 5 | **Placeholder identities for known data** | Letter "M" circles for McDavid/MacKinnon on Today; "ANA"/"BOS" abbreviation circles in Explore Teams — all when real NHL CDN logos are one URL away |
| 6 | **Two stacked gear icons** | One in PuckIQ header (no-op stub), one in Command Center header (customize-modules) — visually redundant |
| 7 | **Heavy paywall scrim** | `rgba(10,14,26,0.55→0.92→0.98)` wall on My Team — content invisible, feels like a dead end |
| 8 | **Empty-state laziness** | Edge segment renders "TEAM EDGE RANKINGS" header with nothing under it; condition was "show if `teamLanding` is non-null" instead of "show if data inside is non-null" |
| 9 | **Data bug rendered to user** | "0th in Atlantic" — `divisionSequence` was null, fallback was `?? 0` instead of computing locally |

## Recommended order for future polish

1. **Wire HeroMatchup into Today** — design exists, component exists, integration is one import.
2. **Build the Settings screen** — once it has a destination, the header gear can come back.
3. **Hub notifications**: inline PRO badge on each toggle so signed-in pros see them as live.
4. **Players hero/elevated fallback** — rank by `points` from the leaders endpoint when categorized data is missing.
5. **Migrate remaining components off legacy `theme.*`** — they all read correctly now (because we re-pointed the values), but the long-term goal is `rinkGlass.*` as the only source. ~30 files.

## Wall-time and process

- **Phase 1** (discovery, plan-mode): 3 parallel Explore agents → routes map + design system audit + plan doc.
- **Phase 2** (walk): 16 screenshots across 5 tabs + 2 hidden + 4 segments + 1 paywall (free state).
- **Phase 3** (vision): committed to **Stat Sheet** without polling, per "loop and build" instruction. Folded in user's "no articles, stats-focused" directive after seeing the Daily Insight module.
- **Phase 4** (build): three waves with simulator re-walks between each. ~25 file edits, mirrored from worktree to main repo so the running Metro bundler picked them up.
